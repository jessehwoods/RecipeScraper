# RecipeScraper

Programs for pulling recipes from recipe sites and then returning them as a JSON. 

#Format

The format of the JSON to be output is as follows:

{
    "name": "Curried Lentils and Rice",
    "ingredients": [
        STRING
    ],
    "steps": [
        STRING
    ],
    "imageURL": STRING,
    "pageURL": STRING
}
